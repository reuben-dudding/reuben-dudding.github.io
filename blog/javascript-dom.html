Control flow and Loops

Control flow is the syntax in which your computer will read and execute your code. It helps to think of it like a step by step guide for your pc to follow and sometimes we have to meet certain conditions in order to progress through the guide. These are called loops, they kind of side track us off our main guide and repeat a line of code and wont continue until a certain condition is met. An example of this could be making your morning coffee. You follow the same steps everyday like:

  1. Put jug on
  2. Get mug and coffee
  3. Put 1 spoon of coffee in mug
  4. Pour hot water in mug
  5. Drink

This would be your control flow but if i wanted to change that and stir after i poured hot water in mug id have a loop that would outline the conditions of how many times i have to stir before i can drink

Accessing data from Arrays and Objects

Arrays and objects are basically containers to store data in to keep things organized. we can all on the data from inside the arrays or objects and use how we like but there are differences in how to access each container of info.

Arrays
We can access the data from arrays using a index system. The index system actually starts from zero so an example could be const burgerIngredients = [cheese, tamatoe, beef, bacon] and how we would call on a specific ingredient would be like burgerIngredients[2] which would call beef since we start counting from zero
Objects
Since objects dont store data like indexes when need to use a dot notation or a bracket notation. first it probably helps to see an object displaying its data so heres a customer's receipt: const receipt ={ items: 'beefBurger' time: '18:56' total: '8:50' } These are key-value pair, the key being the name of the data point, and the value is the data point itself. Now we can use the dot notation or bracket notation to access the data.
