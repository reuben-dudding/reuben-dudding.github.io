What does it mean to display inline vs inline blocks?

The display property is the most important CSS property for controlling layout, as it specifies how an element is displayed.

There are two types of inline tags you can use including display: inline and display: Inline-block, ill be getting into the differences below

Display: Inline
-The display inline property is used to display an element as an inline element like span. Display inline elements are used to change/manipulate elements within text (see what i did there) and are displayed within the line of text. Inline elements include anything that changes a text tag or element within and without breaking the line of text. More examples of inline elements include: This is a strong inline element This is a small inline element This is a span "mini div" inline element This is a sup inline element These tags naturally/by default display inline. Basically, inline elements enable changes to the text within the line of text. No height or width values are affected and only the left and right margins are allowed no top or bottom values.
Display: Inline-block
-Pretty much the same as inline but this displays an element as an inline level block container. The element itself is formatted like disply inline but you are able to manipulate the height and width values to match it up nicely. An example of an display: inline block is the as it displays on a new line and within a block that can be formatted.
Block
-Block properties enable you to start new elements on their own line and takes up the whole width available. You can change the height and width values as you please
