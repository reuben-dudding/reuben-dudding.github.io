What does it mean to display inline vs inline blocks?

^Home :)

The display property is the most important CSS property for controlling layout, as it specifies how an element is displayed.

Display inline
-The display inline property is used to display an element as an inline element like span. No height or width values are affected and only the left and right margins are allowed no top or bottom values.
Inline block
-Pretty much the same as inline but this displays an element as an inline level block container, sort of like a mini div in the middle of another div. the element itself is formatted like disply inline but you are able to manipulate the height and width values to match it up nicely.
Block
-Block properties enable you to start new elements on their own line and takes up the whole width available. You can change the height and width values as you please
